
Estrutura de dados:
- É uma estrutura organizada de dados na memória de um computador ou em qualquer dispositivo de armazenamento, de forma que os dados possam a ser utilizados de forma correta.
- Usando estruturas adequadas através de algoritmos, pode-se trabalhar com uma grande quantidade de dados, como aplicações em bancos de dados ou serviços de busca. 

► Principais estruturas de dados: 
	
• Vetores e Matrizes
- São estruturas de dados simples que podem auxiliar quando há muitas variáveis do mesmo tipo em um algoritmo.
- Vetor é uma variável que armazena várias variáveis do mesmo tipo. É uma estrutura de dados indezada.
- Matriz é um vetor que possui duas ou mais dimensões. 
 
• Registros 
- É uma estrutura qe fornece um formato especializado para armazenar informações em memória. Este recurso permite armazenar mais de um tipo de dado.

• Fila
- Admite remoção de elementos e inserção de novos.

• Lista 
- É uma estrutura de dados que armazena dados de um determinado tipo em uma ordem específica e possuem tamanho ajustável.
- Podem ser: ligadas ou duplamente ligada.

• Pilha
- È uma estrutura de dados que serve como uma coleção de elementos e permite o acesso a somente um item do dado armazenado.

• Arvore
- É uma forma de estrutura de dados que organiza seus elementos de forma hierarquica: RAIZ > NÓS (FOLHAS).

• Tabela Hash
- É uma estrutura de dados especiais, que associa chaves de pesquisa a valores. É uma generalização da ideia de arrays, porém utiliza uma função denominada Hashing para espalhar os elementos, fazendo com que os mesmos fiquem de forma não ordenada dentro do array que define a tabela. 
 
• Grafos 
- São estruturas que permitem programar a relação entre objetos.
